# Competitor Research Skill

Скилл для поиска конкурентов по ключевой фразе и гео, составления таблицы сравнения ДКЦП и вывода рекомендаций для контента сайта.

## Зависимости

- `value-prop-writer` — для понимания нашей ДКЦП и формата инвариантов

## Входные данные

1. **Ключевая фраза** — запрос, по которому ищут продукт
2. **Гео** — регион/город
3. **Наше ценностное предложение** — документ от value-prop-writer

## Результат

- Список конкурентов с URL и основным контентом
- Таблица сравнения по 8 инвариантам ДКЦП
- Рекомендации: что писать на сайте (Hero, Проблема, Возможности, Доверие, FAQ)

## Использование

Обратись к агенту с запросом вроде:

> Найди конкурентов по запросу «конструктор квизов для застройщиков» в Москве. У меня есть наше ценностное предложение [приложи файл]. Составь таблицу сравнения ДКЦП и дай рекомендации для контента сайта.
