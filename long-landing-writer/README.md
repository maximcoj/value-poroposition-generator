# Long Landing Writer

Скилл для создания структуры и текстов большого лендинга на 10-12 экранов.

## Зависимости

- **value-prop-writer** — требуется готовое ценностное предложение с 8 инвариантами и аргументами

## Входные данные

Документ от `value-prop-writer` — файл с инвариантами ценностного предложения и сгенерированными аргументами.

## Выходные данные

Структурированный лендинг на 10-12 экранов с готовыми текстами для каждого блока.

## CTA по лендингу

**3-4 CTA** размещаются в ключевых точках:
1. Hero — первый призыв
2. Решение или Возможности — после «ага-момента»
3. Тарифы или Кейс — после цены и социального доказательства
4. Финальный CTA — в конце

## Структура экранов

| # | Экран | Обязательность |
|---|-------|----------------|
| 1 | Hero | ✅ |
| 2 | Логотипы / доверие | ✅ |
| 3 | Проблема | ✅ |
| 4 | Решение | ✅ |
| 5 | Как это работает | ✅ |
| 6 | Возможности 1 | ✅ |
| 7 | Возможности 2 | Опционально |
| 8 | Кейс / отзыв | ✅ |
| 9 | Тарифы и условия | ✅ |
| 10 | FAQ | ✅ |
| 11 | Финальный CTA | ✅ |
| 12 | Доп. блок | Опционально |

## Файлы

- **SKILL.md** — основные инструкции
- **screen-blocks.md** — детальные шаблоны блоков

## Использование

1. Подготовьте ценностное предложение через `value-prop-writer`
2. Обратитесь к агенту с этим скиллом
3. Предоставьте файл с ценностным предложением
4. Уточните объём: 10, 11 или 12 экранов
