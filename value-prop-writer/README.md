# Value Prop Writer

Скилл для создания ценностных предложений по модели ДКЦП (Дворец Культуры Ценностного Предложения).

## Что делает скилл

Помогает пошагово собрать и проверить 8 инвариантов ценностного предложения:

| # | Инвариант | Вопрос |
|---|-----------|--------|
| 1 | Сегмент | Кто адресат предложения? |
| 2 | Мотивационный конфликт | Почему клиент покупает? |
| 3 | Артефакт | Что мы передаём? |
| 4 | Атрибуты артефакта | Чем продукт особенный? |
| 5 | Вклад клиента | Что клиент вкладывает? |
| 6 | Условия обмена | Как организована сделка? |
| 7 | Атрибуты компании | Почему нам можно доверять? |
| 8 | Ценность | Ради чего всё это? |

После сбора инвариантов скилл:
- Проверяет каждый инвариант по критериям корректности
- Анализирует согласованность между инвариантами
- Генерирует готовые аргументы для коммуникации

## Как использовать

1. Подключите скилл к AI-ассистенту
2. Попросите помочь составить ценностное предложение
3. Отвечайте на вопросы по каждому инварианту
4. Получите готовый документ с аргументами

**Пример запроса:**
> Помоги составить ценностное предложение для моего продукта — конструктора онлайн-курсов для корпоративного обучения

## Пример результата

Скилл выдаёт структурированный Markdown-документ:

```markdown
# Ценностное предложение: [Продукт]

## Целевой сегмент
...

## Мотивационный конфликт
...

## Аргументы ценностного предложения
1. [Аргумент из комбинации инвариантов]
2. ...

## Анализ сходимости
| Проверка | Статус | Комментарий |
|----------|--------|-------------|
| Атрибуты решают конфликт | ✅ | ... |
```

Полный пример смотрите в [example-output.md](example-output.md).

## Критерии качества

Скилл проверяет согласованность инвариантов:

- **Атрибуты ↔ Конфликт** — атрибуты действительно решают проблему клиента?
- **Вклад ↔ Ценность** — вклад соразмерен получаемой ценности?
- **Условия ↔ Конфликт** — условия снижают барьеры входа?
- **Компания ↔ Артефакт** — компания способна доставить продукт?
- **Сегмент ↔ Конфликт** — конфликт характерен для сегмента?
